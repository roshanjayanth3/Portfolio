# Portfolio - Jopaul K Joshy

This is a simple, dark-themed portfolio website built for **Jopaul K Joshy**.

## About
A minimal static site to showcase projects: SecureNotes, KMail (in progress), and Registration Form.

## How to use
1. Extract the ZIP.
2. Serve the folder with a static server (or open `index.html` directly).
   - `python3 -m http.server` (simple local server)
3. Edit `/index.html` and `/assets` to add screenshots and resume.

## Projects included (links are live)
- SecureNotes — https://securenotes-kmpl.onrender.com
- KMail (frontend preview) — https://kmail-frontend.onrender.com
- Registration Form — https://registrationform-37ca.onrender.com
